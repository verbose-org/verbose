# Verbose — comprendre ce qui se passe vraiment

Une série d'articles éducatifs qui utilisent verbose comme support pour apprendre la compilation native, l'assembleur x86-64, le format ELF, et les concepts fondamentaux de l'informatique bas niveau.

Aucun prérequis. Chaque concept est expliqué quand il apparaît.
